## 标题：我写了个开源的 AI Agent 桌面操作系统，能不能超过 LangChain？

---

其实这不是又一个框架。如果大家还记得当年 build-your-own-x 为什么火 —— 因为它重新定义了"学编程"这件事。

我觉得 AI Agent 领域缺的是这个：**一个开箱即用的桌面端。**

现在 GitHub 上的 Agent 项目全是框架（LangChain 136k⭐、AutoGen 58k⭐、CrewAI 51k⭐），全部要写代码才能跑。而 ChatGPT Desktop、Claude Desktop 这种桌面端产品证明了用户有多需要 —— 但它们全是闭源的。

所以我写了个叫 **Kairon** 的开源项目。

### 名字的来历
Kairon 源自希腊语 Kairos（关键时刻）—— Agent 的本质就是在正确时机做正确的事。

### 它能做什么？
- 💬 **开箱即用** —— npm install 完就能聊
- 🤖 **多模型** —— OpenAI、Claude、DeepSeek、甚至本地 Ollama
- 🔧 **内置工具** —— 文件操作、Web搜索、代码执行
- 🔌 **插件系统** —— 像 VSCode 一样扩展
- 🎯 **Agent 编排** —— 可视化拖拽工作流（开发中）
- 💻 **桌面端** —— Tauri + Rust，轻量原生体验

### 技术栈
Tauri + React + TypeScript + Rust，Agent Runtime 自研不依赖 LangChain。

### 当前阶段
MVP 刚成型，核心功能能跑。求 star、求 PR、求各种姿势的贡献。

🔗 https://github.com/tongdaofang/kairon
